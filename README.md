# Cybersecurity Internship Project - DevelopersHub

## Features
- **Input Sanitization:** XSS protection using `validator.js`.
- **Password Security:** Hashing with `bcrypt`.
- **Auth:** Session management via `jsonwebtoken` (JWT).
- **Audit:** Logging security events with `winston`.

## Setup
Run `npm install` and then `node app.js` to see results.
